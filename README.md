# Heart Disease Prediction using Apache Spark ML

## Overview

This project is a healthcare analytics and machine learning application developed using Apache Spark ML and Databricks to predict the likelihood of heart disease based on patient medical attributes. The project demonstrates the integration of big data processing and machine learning workflows in a scalable analytics environment.

## Objective

The primary goal of this project is to analyze patient health data and build a predictive model capable of identifying potential heart disease risk. The project also focuses on understanding how Apache Spark can efficiently process and analyze structured healthcare datasets.

## Features

* Data preprocessing and cleaning using PySpark
* Feature engineering and transformation
* Spark SQL-based analytical queries
* Machine learning model training using Spark MLlib
* Distributed data processing using Apache Spark
* Databricks notebook-based workflow execution

## Technologies Used

* Apache Spark
* PySpark
* Spark SQL
* Spark MLlib
* Databricks
* Python

## Workflow

1. Loaded and explored healthcare dataset using Spark DataFrames
2. Cleaned and transformed raw medical data
3. Performed feature engineering and analytical queries using Spark SQL
4. Built machine learning pipelines using Spark MLlib
5. Trained and evaluated predictive models for heart disease classification

## Skills Demonstrated

* Big Data Analytics
* Machine Learning
* Distributed Computing
* Data Preprocessing
* Feature Engineering
* Predictive Analytics
* Spark-based ML Pipelines

## Outcome

The project demonstrates how scalable machine learning workflows can be used in healthcare analytics to assist in early disease risk prediction and data-driven decision-making.
